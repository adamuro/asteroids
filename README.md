# Asteroids

### How to run the game?

1. Copy the repository:  
  ``` git clone git@github.com:adamuro/asteroids.git ```

2. Download SFML libraty:
  ``` sudo apt-get install libsfml-dev```

3. Create the executable file:  
  ``` cd asteroids ```  
  ``` make ```

4. Run the game  
 ``` ./asteroids ```
 
 ### How to play?
 
 * W - accelerate
 * A and D - rotate
 * Space - shoot
